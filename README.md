# SDLXLIFF File Processor

A web application for processing SDLXLIFF translation files, converting MT segments to interactive status.

## Features
- Convert `origin="mt"` to `origin="interactive"`
- Clean XML structure
- User-friendly interface
- Download processed files

## How to Use
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
3. Access via browser at `http://localhost:8501`

## Deployment
To deploy on Streamlit Cloud:
1. Fork this repository
2. Go to [Streamlit Cloud](https://share.streamlit.io/)
3. Connect your GitHub account
4. Select this repository and branch
5. Click Deploy!
